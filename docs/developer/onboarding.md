## Quick reference

- **Prerequisites:** Node.js 20+, Python 3.11+, Docker, Git
- **Setup:** `cp .env.example .env` then start backend (`cd backend && python -m app.main`) and frontend (`cd frontend/web && npm run dev`)
- **Web:** http://localhost:3000  
- **API docs:** http://localhost:8000/docs

See [ONBOARDING.md](../ONBOARDING.md) for detailed steps, project structure, and runbooks.
